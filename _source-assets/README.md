# Source Assets

This directory stores original and print-ready artwork that must remain in the repository but should not be deployed as public website assets.

GitHub Pages uses Jekyll for this repository. Jekyll excludes directories whose names begin with an underscore, so `_source-assets/` is not published.

## Contents

- `originals/` - high-resolution profile, logo, social image, Google profile card, and portfolio source images
- `print/portfolio/` - print-resolution business card artwork
- `editable/portfolio/` - original social post artwork

Delivery-ready, actively referenced website images remain in `images/`.
